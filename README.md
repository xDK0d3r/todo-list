# 📝 To-Do List (CLI)

A command-line To-Do List application built with Python that helps users organize and manage daily tasks through a simple, interactive, menu-driven interface.

>[!Note]
>This project was created as part of my Python learning journey to practice writing clean,      modular code and implementing core CRUD (Create, Read, Update, Delete) operations.

## ✨ Features

* ➕ Add new tasks
* 📋 View all tasks
* ✏️ Edit existing tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 📌 Display completed and pending tasks
* 💾 Save tasks to a file
* 📂 JSON-based task storage
* ✔️ Input validation for a better user experience
* 🖥️ Menu-driven command-line interface

## 🛠️ Technologies Used

* Python 3
